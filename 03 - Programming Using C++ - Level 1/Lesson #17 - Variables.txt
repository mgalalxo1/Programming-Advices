✅ Valid Variable Names:
Age
✔ Valid – Starts with a letter, contains only letters.

My_Age
✔ Valid – Letters and underscore are allowed.

_Age
✔ Valid – Variables can start with an underscore.

X2
✔ Valid – Starts with a letter and includes numbers, which is allowed.

MyName
✔ Valid – Letters only, good naming convention.

❌ Invalid Variable Names:
My-Age
❌ Invalid – Hyphens (-) are not allowed in variable names. It's interpreted as subtraction.

2x
❌ Invalid – Variable names cannot start with a number.

My Name
❌ Invalid – Spaces are not allowed in variable names.

#Name
❌ Invalid – Special characters like # are not allowed in variable names.

$Name
❌ Invalid in most languages (like C++, Java, Python) – $ is not commonly allowed unless in specific languages like JavaScript (even then, not recommended).

N@me
❌ Invalid – @ is a special character and not allowed.

int
❌ Invalid – int is a reserved keyword in many programming languages like C++, Java, and Python.

float
❌ Invalid – float is also a reserved keyword used to define data types.

Summary of Rules for Valid Variable Names:
Must start with a letter or underscore.
Can contain letters, digits, and underscores.
Cannot start with a digit.
No spaces or special characters (like @, #, -, $, etc.).
Cannot use reserved keywords as variable names.



#include <iostream>
using namespace std;

int main() {
    string name = "Mohammed Abu-Hadhoud";
    int age = 44;
    string city = "Amman";
    string country = "Jordan";
    int monthlySalary = 5000;
    int yearlySalary = 60000;
    char gender = 'M';
    bool married = true;

    cout << "Name    : " << name << endl;
    cout << "Age     : " << age << " Years." << endl;
    cout << "City    : " << city << endl;
    cout << "Country : " << country << endl;
    cout << "Monthly Salary : " << monthlySalary << endl;
    cout << "Yearly Salary  : " << yearlySalary << endl;
    cout << "Gender : " << gender << endl;
    cout << "Married: " << married << endl;

    return 0;
}
